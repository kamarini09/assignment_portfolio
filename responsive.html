13C.02.01 Responsive design

This assignment focuses on designing web pages that are optimized for different screen sizes and devices by using CSS media queries and flexible layouts. We learned how to create a responsive grid system, adjust font sizes and images, and ensure that their web pages are accessible and easy to use on a variety of devices. By completing this assignment, we gained valuable skills in creating flexible and adaptable web pages that can provide a seamless user experience across multiple devices. The ability to design responsive web pages is essential in today's digital landscape where users access websites and applications from a variety of devices with different screen sizes and resolutions.
