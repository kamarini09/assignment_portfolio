13C.01.02 Working with colors and animations

This assignment focuses on enhancing the visual appeal and user experience of a web page by learning how to use color palettes and CSS animations effectively. We gained valuable skills in choosing appropriate color schemes, using CSS transitions and keyframes, and creating subtle yet visually pleasing animations. By completing this assignment, we are able to make our web pages stand out by using color and animation to capture users' attention and improve their overall experience..
