13c.01.01 Fonts and Components

Designing a visually appealing and functional web page requires a deep understanding of font hierarchy and UI components. In this assignment, we learned how to create an effective font hierarchy using one or two fonts and apply it to a web page. We also gained valuable skills in improving UI components by following guidelines for designing UI components and CSS techniques covered in class. By completing this assignment, we have a better understanding of font sizes, line height, margins, and padding, as well as how to use CSS flex to layout cards and create subtle borders, rounded corners, and shadows. This assignment was an essential step towards creating visually stunning and functional web pages.
